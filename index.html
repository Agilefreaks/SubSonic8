Welcome to Subsonic8.

Subsonic8 is an open source Windows 8 client for the popular media streaming server Subsonic. To be able to use this application you have to have an account on a subsonic server. You can get the server application over at: http://www.subsonic.org.

Features

Some of the features of the Subsonic8 client are:

  • Play audio files
  • Play video files
  • Search through your media library
  • Media playback integration with Windows (hotkeys, volume, notifications)
  • Save/Load playlists to your hard drive or directly to Subsonic
  • Manage your remote playlist
  • Pinning the app to your start screen will give you a live tile showing the currently playing song and it's cover art
  • Secure credentials storage using Windows Password Vault
  • Playlist Filtering
  • A friendly and easy to use Snapped View for the Playback Screen
